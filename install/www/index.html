VisionEval Test Installer

Through this page, you can:

  • Find links to the VisionEval git repository:
    • Test installer repository (powering this site)
    • Release repository
    • Pooled fund development repository
    • AASHTO/ODOT development repository
  • Before anything else, download and run the Windows installer for the appropriate version of R
  • Download and run one of the experimental installers:
    Online installer
    Use if your R installation can access repositories through the internet (any OS/architecture, R 3.5.1)
    Windows installer
    Download this version for a ready-to-run Windows installation that requires no internet connection
    • Offline source installer
    (not available yet) Download this version for a cross-platform installer that requires no internet connection
To use one of the installers:
  1. Download it
  2. Unzip it into an empty folder
  3. Then either:
    • Double-click Install-VisionEval.bat from that folder
    or if you have multiple versions of R installed, and the VisionEval version is not primary, or if you're using the Online installer on a non-Windows architecture.
    1. Start R
    2. Use File / Change dir... to navigate to the folder where you unzipped the installer
    3. Enter the R function call: source("Install-VisionEval.R")
  4. After that, you can sort of use the Getting Started instructions, or use the convenience functions vegui(), verpat() or verspm()

If you're stumped, get in touch with "jeremy R" at one of the usual places.

The online installer uses this repository: https://visioneval.jeremyraw.com/R/
