VisionEval Test Installer

Through this page, you can:

  • Find links to the VisionEval git repository:
    • Test installer repository (powering this site)
    • Release repository
    • Pooled fund development repository
    • AASHTO/ODOT development repository
  • Download the Windows installer for an appropriate version of R
  • Download and run one of the experimental installers:
    Online installer
    Use if your R installation can access repositories through the internet
    Offline installer
    Download this version if your R installation cannot get on the internet
To use one of the installers:
  1. Download it
  2. Unzip it into an empty folder
  3. Then either:
    • Double-click VisionEval.Rdata from that folder
    or (if you have multiple versions of R installed, and the VisionEval version is not primary)
    1. Start R
    2. Use File / Change dir... to navigate to the folder where you unzipped the installer
    3. Use File / Load workspace... to load VisionEval.Rdata
    4. Enter the R function call: visioneval()

If you're stumped, get in touch with "jeremy R" at one of the usual places.

The online installer uses this repository: https://visioneval.jeremyraw.com/R/
